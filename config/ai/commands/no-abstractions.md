# NO ABSTRACTIONS

## Instructions

This command requires all code to be written with **zero abstractions**.

### What NOT to do

- Do NOT create abstraction layers (functions, variables, classes, methods, etc.)
- Do NOT worry about reusability or code organization
- Do NOT create shared logic files or utility functions
- Do NOT refactor for cleanliness

### What TO do

- Inline all code directly where it's needed
- Duplicate code if necessary, don't abstract it
- Write everything in one place, as linearly as possible
- Prioritize straightforwardness over reusability
